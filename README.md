# mini-compiler
This project is a mini compiler implemented in C that evaluates mathematical expressions entered by the user.
It supports basic arithmetic operations, parentheses handling, exponentiation, and several mathematical functions.

The program uses a recursive descent parsing technique to correctly interpret and evaluate expressions according to operator precedence.

Features:

Basic arithmetic operations
Addition (+)
Subtraction (-)
Multiplication (*)
Division (/)
Advanced operations
Power operator (^)
Mathematical functions
sqrt(x) – Square root
log(x) – Base 10 logarithm
ln(x) – Natural logarithm
Expression validation
Parentheses support
Floating-point number support
Division by zero error handling

Technologies Used:

C Programming Language
Standard Libraries
<stdio.h>
<math.h>
<ctype.h>
<string.h>
